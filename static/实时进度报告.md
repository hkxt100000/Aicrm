## 🎨 设计系统特性

### 颜色系统
```css
主题色: #007AFF (iOS 蓝)
辅助色: #5856D6 (紫色)
成功色: #34C759 (绿色)
警告色: #FF9500 (橙色)
危险色: #FF3B30 (红色)
灰度: 10级 (#FAFAFA → #1C1C1E)
```

### 字体系统
```css
字体族: SF Pro Display / PingFang SC
字号: 11px → 40px (9级)
字重: 300 → 800 (6级)
```

### 圆角系统
```css
xs: 4px  (小标签)
sm: 6px  (小按钮)
md: 10px (标准按钮、卡片)
lg: 14px (大卡片、弹窗)
xl: 20px (欢迎卡片)
```

### 阴影系统
```css
sm: 0 1px 3px      (卡片)
md: 0 4px 12px     (按钮悬停)
lg: 0 8px 24px     (卡片悬停)
xl: 0 16px 48px    (弹窗)
2xl: 0 24px 64px   (模态框)
```

## 🔧 技术实现

### CSS 架构
```
1. apple-design-system.css   (基础变量和工具类)
   ↓
2. style.css                  (原有样式)
   ↓
3. apple-style-overrides.css  (Apple 风格覆盖)
```

### 优势
- ✅ **非破坏性** - 保留原有样式作为基础
- ✅ **渐进式** - 逐步覆盖，可随时回退
- ✅ **可维护** - 使用 CSS 变量，易于调整
- ✅ **暗黑模式** - 自动跟随系统设置
